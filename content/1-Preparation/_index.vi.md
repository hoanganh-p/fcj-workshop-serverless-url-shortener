### 1. Công cụ cần thiết
Trước khi bắt đầu, hãy đảm bảo máy tính của bạn đã cài đặt:
*   **Visual Studio 2022** hoặc **VS Code**.
*   **.NET 8 SDK**.
*   **Node.js** (v18 trở lên) và **npm**.
*   **AWS CLI** (đã cấu hình `aws configure` với Access Key).
*   **Postman** để test API.

### 2. Tải mã nguồn
Clone source code dự án từ repository:
```bash
git clone https://github.com/hoanganh-p/url-shortener-backend.git
```
```bash
git clone https://github.com/hoanganh-p/url-shortener-frontend.git
```
